## 二分法（binary）

易错点：

- r 是取n还是n-1

- 循环条件 l <= r  还是 l  < r

- 左右指针的移动 ，取m、m+1或m-1

左闭右开[ l , r ), 因为不取所以r取n

```python
def bisnary_search(arr,target):
    # 假设区间为左闭右开 [ l , r )
    l = 0
    r = len(arr)
    # 为保证区间内整数不为空，l<r
    while l < r:
        m = r + (r-l)//2        # 虽然pyhton不会int溢出，但是为了规范，要写
        if target <= arr[m]:
            r = m               # 右开，取m
        elif arr[m] < target:
            l = m+1             # 左开，arr[m]被排除，取m+1
    # 循环结束的条件 l=r，此时arr[m] < target < arr[l]
    # 返回第一个大于等于target的位置,使用l、r都可以，
    # 但语义为target的值在左侧 target <= arr[ r , len )
    return l  
```
