# 子串或数组

## 3.无重复最长子串

给你一个字符串s ，请你找出一其中不含有重复字符的最长子串的长度。

### 思路

用字典记录最近字符最近一次出现的坐标，如果 arr[right] 在字典里面，查看dict[char]是否在窗口内，在说明重复，不在则更新.
由于使用的是字典，删除操作开销比较大，所以需要进行判断是否在窗口内的操作，而不是删除键
优化：可以利用长度等于字符集长度的数组作为数据结构，在对应的下标进行+-1操作即可实现删除和添加了

## 53. 最大(连续)子数组和

给你一个整数数组 nums ，请你找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

### 思路

1. 动态规划
   思考思路：题目要求的是连续的数组，找[l,r]区间的最大和的连续子数组比较难，因为无法确定该数组的两端，那我们退而求其次，求以l或以r为端点的连续数组则比较容易求。
   dp[i] 记录以i结尾的最大子数组和。dp[i+1] = (dp[i] + nums[i] ,nums[i])
2. 贪心
   做题时，可以想到利用前缀和，遍历所有的[l,r]区间，但时间为O(n^2)
   优化：假设[l,r]区间为最大和：Sum[l,r] = pre[l+1] - pre[r]; 利用极限思想，在l<=r的限制条件下，pre[l+1]必然是取最大值，pre[r]必然是取最小值。所以在遍历过程中记录前面的最小前缀和,与当前前缀和的差值就是[0,r]（以r为结尾的最大和）
3. 线段树(思维扩展)
   适用于存在数据更改和多次查找任意区间的需求的情况。
   [l,r] 左子区间[l,m] 右子区间[m+1,r] 
   要维护以下变量：
   以l为开头的最大和    lSum        ：max(ls_lSum,ls_total_Sum + rs_lSum)  
   以r为结尾的最大和    rSum
   区间数组的求和Sum    total_Sum
   区间的最大和         ans_Sum     ：max(ls_ans_Sum, rs_ans_Sum, ls_rSum + rs_lSun)

   lSum(rSum同理):
   两种情况，lSum对应的区间没有跨域m——即ls_lSum;lSum对应的区间跨域了m——即[l,m+i]，ls_total_Sum + rs_lSum
   ans_Sum:
   分三种情况，没有跨区：ls_ans_Sum ； rs_ans_Sum ； 跨区：ls_rSum + rs_lSum
