# 做题技巧

## 切片操作
1 描述
sequence[start:stop:step]

start：起始索引（默认开头）
stop：结束索引（默认末尾）
step：步长（可为负数）

返回的是新序列（注意内存使用）

2 适用类型
| 类型 | 示例 | 结果 |
|------|------|------|
| 字符串 | `'abc'[::-1]` | `'cba'` |
| 列表 | `[1,2,3][::-1]` | `[3,2,1]` |
| 元组 | `(1,2,3)[::-1]` | `(3,2,1)` |
| bytes | `b'abc'[::-1]` | `b'cba'` |

3 适用场景

- 安全反转与倒序遍历
original = [1, 2, 3]
reversed_copy = original[::-1]  # original 不变
list.reverse()会节省内存会修改原序列，而且只能用在列表类型上

- 回文判断、判断是否对称
s == s[::-1]

- 取后 N 个元素（倒序或正序）

- 每隔一个取一个（奇偶位分离）
