- 二维列表

matrix = [[(i, j) for j in range(列)] for i in range(行)]
matrix[行][列]
```pyhton
# 定义行数和列数
m = 5
n = 2
# 创建二维列表

matrix = [[(i, j) for j in range(n)] for i in range(m)]

for row in matrix:
    print(row)
print(matrix[0][1])
>>>
[(0, 0), (0, 1)]
[(1, 0), (1, 1)]
[(2, 0), (2, 1)]
[(3, 0), (3, 1)]
[(4, 0), (4, 1)]
(0, 1)
```
